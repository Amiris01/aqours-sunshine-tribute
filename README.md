# Aqours — Love Live! Sunshine!! Fan Tribute

A single-page fan tribute to **Aqours**, the nine-member school idol group from
*Love Live! Sunshine!!*. Built with Vite + React.

🔗 **Live:** https://amiris01.github.io/aqours-sunshine-tribute/

## Getting started

```bash
npm install
npm run dev      # dev server
npm run build    # production build -> dist/
npm run preview  # preview the build
```

On Windows you can also double-click **`start.bat`**.
