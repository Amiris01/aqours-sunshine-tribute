# Design archive

Reference material the current app (`/src`) was built from. **None of this is
part of the build** — it's kept for historical context and as the source of
truth for exact spacing, motion, and content.

- **`HANDOFF-BRIEF.md`** — the original handoff brief describing the intended
  page (the long-form spec; the app was implemented from this).
- **`DESIGN-HANDOFF.md`** — shorter redesign brief for the visual refresh.
- **`redesign-snapshot/`** — the original Design Component HTML prototype
  (`Aqours-Sunshine.dc.html` + `preview/`) and an early `src/` snapshot.
  Authored in a proprietary `<x-dc>` / `support.js` format; will not run as
  plain HTML outside that environment.
